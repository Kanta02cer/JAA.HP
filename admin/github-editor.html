GitHub直接編集ガイド

最も簡単で確実なコンテンツ編集方法

📝 方法1: GitHub Webエディタ

1GitHubリポジトリにアクセス

📁 リポジトリを開く

2編集したいファイルをクリック

3鉛筆アイコン（編集）をクリック

4内容を編集

5「Commit changes」で保存

# 編集可能なファイル _news/2025-01-20-test-article.md _news/2025-10-01-press-release.md index.html about.html news.html
🔧 方法2: 新しい記事の作成

1GitHubリポジトリで「Add file」→「Create new file」

2ファイル名を入力（例: _news/2025-01-25-new-article.md）

3以下のテンプレートを使用

--- layout: news-detail title: "新しい記事のタイトル" description: "記事の説明" date: 2025-01-25 category: "お知らせ" tags: [] image: "" keywords: "" author: "日本学生アンバサダー協会" draft: false --- ここに記事の内容を書きます。 ## 見出し1 段落の内容... ## 見出し2 - リスト項目1 - リスト項目2 - リスト項目3
⚡ 方法3: ローカル開発環境

1リポジトリをクローン

git clone https://github.com/Kanta02cer/JAA.HP.git cd JAA.HP

2お好みのエディタで編集

3変更をコミット・プッシュ

git add . git commit -m "記事を更新" git push origin main
🎯 推奨方法

初心者向け: 方法1（GitHub Webエディタ）

上級者向け: 方法3（ローカル開発環境）

緊急時: 方法1（GitHub Webエディタ）
